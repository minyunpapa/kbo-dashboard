# ⚾ SSG 랜더스 2026 대시보드

**https://minyunpapa.github.io/kbo-dashboard/**

KBO 2026 시즌 SSG 랜더스 팬 대시보드. 네이버 스포츠 공개 API에서 시즌 전 경기를 수집해
정적 페이지로 렌더링한다. 서버 없음 — `data.js` 하나로 동작.

## 기능

- **개요**: 순위·성적 요약, 순위 추이 차트(10팀), 팀 순위표, 최근 10경기, 상대전적, 월별 성적
- **다음 경기**: 선발 예고 매치업(양팀 선발 시즌 성적 + 상대전적), 라인업 발표 시 자동 표시
- **일정·결과**: 시즌 게임 로그 → 클릭 시 박스스코어(라인스코어·승패세 투수·주요 기록·타자/투수 라인)
- **선수단**: 타자/투수 시즌 집계 테이블(정렬 가능) → 클릭 시 선수 상세 + 최근 5경기

## 자동 갱신 (GitHub Actions)

| KST | 목적 |
|---|---|
| 09:30 | 선발 예고 |
| 13:10 (주말) / 16:10 / 17:40 | 라인업 발표 반영 |
| 23:00 | 경기 결과·박스스코어 |

변경이 있을 때만 `data.js`를 커밋 → GitHub Pages 자동 재배포.

## 수동 실행

```bash
python3 collect.py   # 의존성 없음 (표준 라이브러리만)
open index.html      # file:// 로도 동작
```

## 구조

- `collect.py` — 수집기: 캘린더 스윕 → 경기 요약 → SSG 박스스코어 → 선수 집계 → 선발/라인업 프리뷰
- `data.js` — `window.KBO_SEASON = {...}` (수집 산출물)
- `index.html` — 대시보드 (Chart.js + Pretendard, 단일 파일)
- 시범경기 제외: 공식 순위와 누적 전적이 일치하는 개막일을 자동 탐색
